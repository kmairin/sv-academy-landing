> The Program

5 Saturdays.
1 real app.

Every Saturday 9:00 AM – 12:00 PM at the Lertlah School Apple Computer Lab. Pre-configured machines, professional environment — no laptop required.

📋
Week 1

The Blueprint

Meet the tools (Claude Chat, Stitch, Claude Code, Vercel, GitHub). Learn what a prompt is, how AI works, and key terminology. Build a template app together. Then start YOUR app — write a PRD with KPIs and design your first UI.

Template app deployed · PRD complete · V1 design started

🛠️
Week 2

Debug & Polish

Quiz on Week 1 concepts. Build your app — design, code, preview, fix. Install analytics and error tracking. Push to GitHub. Deploy to Vercel. Your app is now LIVE.

App deployed with live URL · analytics installed · code on GitHub

🚀
Week 3

Ship & Observe

Peer critique session — learn constructive feedback, present your app, get real comments. Improve based on feedback. Update your PRD. Add automated tests. Learn what 'regression' means and why it matters.

Improved app · updated PRD · test cases added

⚡
Week 4

Feature Sprint

Analyze real user data — page views, latency, daily users, error logs, web vitals. Build new features based on data and user feedback. Prepare your 5-slide pitch deck for Demo Day.

New feature shipped · pitch deck ready

🎤
Week 5

The Keynote

Final polish. Parents arrive at 10:30. Each student pitches their app: 3 minutes + 2 minutes Q&A. Certificates awarded. You are now an app builder.

Pitch delivered · certificate earned · app live

> What You'll Learn

What You'll Actually Learn

01

Product Thinking

Plan what to build, why you're building it, and who will use it. Write a PRD like a real product manager at a tech company.

02

AI Literacy

Use AI as a real building tool — not just a toy. Learn to prompt, direct, and iterate with AI to create production-quality apps.

03

>_ building
> The Workflow

Brainstorm. Design. Build. Ship.

01
Claude Chat
Brainstorm + PRD

Chat with Claude to brainstorm your app idea, define features, write a PRD with KPIs, and debug problems along the way.

02
Stitch
Design

Turn your idea into a pixel-perfect UI with Google's AI design tool. Describe what you want, refine until you're happy — no Figma skills needed.

03
Claude Code
Build

AI pair-programs with you to turn designs into a working app. NextJS frontend, NestJS backend, TypeScript — real production code.

04
Vercel
Deploy

Ship your app to the internet with one click. Real URL, real users. Auto-deploys every time you push to GitHub.

Powered by Claude + Stitch + Vercel · All tools provided in-class

NextJS · NestJS · TypeScript · Postgres · R2 · Redis · PostHog · Sentry

Real Infrastructure,
Not Just Syntax

Every student works with production-grade tools — NextJS, NestJS, TypeScript, Postgres, PostHog analytics, and Sentry error tracking. The same stack used by real startups.

From Idea to Live App

Claude Chat for brainstorming, Stitch for UI design, Claude Code for building, Vercel for deployment — a complete pipeline where each step flows naturally into the next.

Demo Day:
Pitch Like a Founder
